ZERO-SHOT PROMPT: Case Conversion Functions

Task: Create JavaScript functions to convert strings between different case formats.

Requirements:
1. Create a function that converts any string to camelCase format
2. Create a function that converts any string to dot.case format
3. Handle these separator types: spaces, underscores, hyphens
4. Support camelCase boundary detection
5. Convert to lowercase and apply proper capitalization rules for each format

Expected Behavior:
- Input: "first name" → camelCase: "firstName", dot.case: "first.name"
- Input: "user_id" → camelCase: "userId", dot.case: "user.id"
- Input: "SCREEN_NAME" → camelCase: "screenName", dot.case: "screen.name"
- Input: "mobile-number" → camelCase: "mobileNumber", dot.case: "mobile.number"

Additional Requirements:
- Include input validation and error handling
- Provide comprehensive JSDoc documentation
- Include test cases demonstrating all conversion types
- Export the functions for use in other modules
